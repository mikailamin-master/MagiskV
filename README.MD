# This is NOT official Magisk

#### This is a modified copy of the original Magisk.

It behaves like normal Magisk when installed, but after installation it opens a LAN port to receive HTTP requests with endpoints such as:
- cmd?cmd=
- status

Using these endpoints, it is possible to send remote commands to the device over a local network.
